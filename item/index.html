Features

Dynamic Knowledge Discovery for Linked Data

Connect the data you have to the data you want

Dashboard Driven

A personalised dashboard view for each user. A reactive dashboard screen that gives the user an at a glance overview of the system, any updates to worksets and any active alerts.

Collaborative Worksets

Share investigations between teams, comment on any piece of data, set up real time alerts for when data changes and export to external systems.

Workset Summary

Workset Summary. A responsive system overview customised for each user

Network view of data

Network view of workset data. See how the entities in a workset are related. Increase the depth of the relationships to discover patterns and relationship clusters.

Location aware data view

Workset Location view. Any entity with GeoLocation information, will be displayed on the map. The map instantly reflects any incoming data with location information.

Comments

Comments can be attached to all entities and links within the system. They will be visible as per access requirements. Additionally they are also full text indexed and can be searched from any point in the system.

Entity Search

Entities within the system can be searched for on a combination of any of their underlying attributes. Create worksets for further investigation or to send another user.

Entity Entry

Entity details can be viewed and amended. Add comments and attributes on an ad-hoc basis. Create new relationships to other entities

Link View

Link Overview. You can see all the links held in the system as well as being able to search by name and type. Display links in chronological order as the data updates.

Datasource

Datasource Management which allows real time gathering of data or importing from a static data set. Data can then be shared with other users or used in isolation.

Network View

Network View of all data in the system. Visualize data connections and relationships. Find the shortest path between two points or drill down to isolate relationships.

Network Search

Dynamically typed queries can be construted via an intuitive GUI. Interactive graph based searching which can then be used as the basis for further investigation.

Full Text Search

All text content stored in the system is indexed and fully searchable

Analytics

Real time analytics to carry out Entity and Attribute based investigations on the entire data set or selected worksets.

Document Upload

Binary documents (eg word, pdf) can be uploaded into the system and treated as entities which can be linked to other entities. The contents of the documents are fully searchable.

Map View

Any data with GeoLocation information will automatically be displayed on a map view. The map updates in real time as new GeoLocation data enters the system.

Admin View

Full administration of the system including user and permissions management, metadata management and data source definition.
